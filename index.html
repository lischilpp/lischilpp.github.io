Bachelor Thesis: Phoneme classification and alignment through recognition on TIMIT

In this work, we explore a hybrid approach combining Artificial Neural Networks (ANNs) and Dynamic Time Warping (DTW) for phoneme alignment on the TIMIT dataset. The method leverages the output probabilities from a neural phoneme recognition model alongside a probability-based DTW to achieve precise phoneme alignment. For phoneme recognition, our model achieves an 18.1% Frame Error Rate (FER), which represents a 4.0% improvement over the state-of-the-art. Our alignment method attains an 86.3% phoneme boundary accuracy within a 20ms tolerance. To efficiently train the network we introduce the CyclicPlateauScheduler, a learning rate scheduler that combines triangular cyclic learning rates with ReduceLROnPlateau.

TEQST

TEQST is a user-friendly WebApp designed to simplify the creation and management of speech data. It features a modern, platform-independent web interface that ensures seamless access across various devices. The account management system supports different permissions for publishers, readers, and listeners, providing tailored access and control. Users can create multi-level folder structures to efficiently organize texts, upload texts and automatically split them into separate parts, and easily share texts with readers using filter options. TEQST also offers comprehensive statistics for user completion and the ability to download recorded texts as audio files, making it an invaluable tool for distributing text and collecting speech data.

Unity Voxel Engine

A voxel engine developed using the Unity3D game engine.
It is written in C# and offers an immersive and interactive environment for voxel-based worlds. It features fixed size spherical chunk loading for optimized rendering and seamless navigation. The engine supports transparent materials like glass and connected blocks such as water, along with a variety of block types to create diverse terrains and structures. It includes terrain generation from noise to ensure unique and dynamic landscapes. Players can interact with the environment through intuitive block placement and destruction using right and left-click controls, and can quickly modify large areas with the fast block destroying feature by holding the 'F' key.

Math Plotter

MathPlotter is a graphing tool designed to simplify the plotting of objects in a parallel perspective 3D coordinate system. This program is ideal for visualizing points, objects, and vectors as seen in math schoolbooks. Key features include zoom-in, zoom-out, and move functionalities for easy navigation, along with labeled axes for clarity. The tool incorporates a math notepad-like square grid and supports labeled vectors, rectangles, circles, cuboids, and tetrahedrons. Additionally, it can plot 1D and 2D functions, making it a versatile and user-friendly tool for educational and professional use. The tool is implemented using JavaScript and the HTML5 Canvas element, ensuring a responsive and interactive user experience directly within the web browser.

The Apocalypse

"The Apocalypse" is an immersive art project created using the Unity 3D engine and C#, combining animation and an interactive environment to depict a dramatic narrative of the moon crashing into the Earth. Key features include a Dolly Zoom on a TV screen with a flip book animation recounting the story of the moon landing, and an intense animation of a rocket crashing into the moon with extensive flames. In the 3D landscape, users can walk around, explore, and experience the end of the world as the moon descends onto the Earth, filling the scene with flares until everything fades to white.

Marching Cubes Planet

Marching Cubes Planet is a dynamic and interactive simulation created using the Unity 3D engine and C#. It features smooth voxel terrain generated with the marching cubes algorithm, providing a seamless and visually appealing landscape. The planet has point gravity directed towards its center, enhancing the immersive experience. Efficient rendering is achieved through chunk loading, ensuring smooth performance. Users can interactively walk around the planet and modify the terrain by "melting" it with a mouse click, adding a unique interactive element to the exploration.
